Nous allons voir dans cette vidéo comment fonctionnent les vues listées dans Odoo. Ici, par exemple, dans l'application Contact ou dans n'importe quelle autre application que vous ouvrez dans Odoo, vous avez la possibilité en haut à droite de sélectionner les vues listées. Dans ces vues, vous aurez chaque fois la possibilité de sélectionner ici un certain nombre de champs ou d'en cacher d'autres. Par exemple, ici on va enlever les activités et rajouter le téléphone mobile. Il est possible aussi sur la majorité des champs de les trier par ordre alphabétique ascendant ou descendant. Si vous souhaitez voir l'ensemble des enregistrements en une seule fois, vous pouvez cliquer sur le bouton de la colonne. Par défaut, un certain nombre d'enregistrements sont affichés. Dans mon exemple, seulement vingt contacts sont affichés.

Pour afficher les enregistrements suivants, il vous suffit de cliquer sur les flèches de droite ou de gauche pour aller sur les pages suivantes ou précédentes. Si vous souhaitez voir l'ensemble des enregistrements en une seule fois, vous pouvez cliquer sur le bouton de la colonne. Vous pouvez aller sélectionner ici le chiffre, rajouter un 0 par exemple pour dépasser le nombre total d'enregistrements, appuyer sur Entrée et maintenant l'ensemble des 57 enregistrements sont affichés. Sur une vue liste, vous avez la possibilité de sélectionner un certain nombre d'enregistrements qui vont vous donner accès à des actions de masse. Vous pouvez par exemple en une fois, en cliquant sur l'action, archiver l'ensemble des 5 contacts qui sont sélectionnés. Suivant les applications, différentes actions vous seront proposées. Par exemple, en comptabilité, vous pourriez envoyer en une seule fois plusieurs factures de cette manière.

La sélection multiple vous permet aussi de mettre à jour massivement des données. Vous pouvez par exemple mettre à jour des données de la même façon. Vous pouvez aussi mettre à jour le vendeur de ces 5 contacts en une seule fois. Cette fonctionnalité est à utiliser avec précaution, comme vous pouvez vous l'imaginer. Dans les applications en sens large, vous avez donc cette vue liste qui est très utilisée. Il y a aussi un certain nombre d'autres vues qui sont disponibles et que nous pourrons voir plus tard.
